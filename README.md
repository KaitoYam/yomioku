# Yomioku

## プロジェクト概要

「Yomioku」は、いつ何を見たのかを記録するためのリポジトリです。
記録対象は技術関連の書籍や動画になります。

対象期間は 2025 年 2 月 10 日からのものになっています。※現状

## 目標と目的

学習のモチベ向上

## ディレクトリ構造

```
├── video/
│   ├── 見た動画タイトル.md
├── book/
│   └── 読んだ書籍名.md
└── README.md
```

## 記録方法

1. 読んだ書籍や動画（シリーズ単位）と同じ名前のファイルを作る
2. テンプレートに沿って内容を埋める
3. 学習済みの項目に[x]を入れる

## ブランチ

### main

- マスターブランチ

### YYYYMM

- 月ブランチ
- 月の初めに作成し、月の終わりに main ブランチにマージする

### YYYYMMDD

- 日にちブランチ
- その日の初めに作成し、終わりに月ブランチにマージする
